## 主流浏览器内核

- IE: Trident
- Firefox: Gecko
- Safari: Webkit
- Chrome: Webkit --> Blink
- Opera: Presto --> Blink

### `user agent`, `UA`, 用户代理（浏览器）

### `-webkit-` 前缀

`-webkit-` 前缀是 Webkit 内核的私有前缀，用于区分 Webkit 内核的 CSS 属性和其他浏览器的 CSS 属性。
